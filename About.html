About the System

The Smart Refrigerator Monitoring System (SRMS) is a professional-grade IoT solution designed to safeguard perishables through real-time environmental telemetry and intelligent inventory tracking.

rocket_launch Project Overview

SRMS was conceived to solve the critical challenge of food waste and safety in modern households and commercial kitchens. By leveraging precise sensor arrays, we provide users with high-fidelity data regarding their cold-storage environments.

  • check_circle Real-time temperature & humidity monitoring with threshold alerts.
  • check_circle Automated door-state detection to prevent thermal leakage.
  • check_circle Historical trend analysis for equipment performance auditing.
  • check_circle Seamless cloud integration for remote access anywhere.

Hardware Ecosystem

memory

ESP32 Microcontroller

The brain of the system. Dual-core processing with integrated Wi-Fi and Bluetooth for low-latency data transmission.

Core Controller
device_thermostat

DHT11 Sensor

High-precision digital sensor providing consistent temperature and relative humidity readings every 2 seconds.

Environment Monitoring
door_open

Door Reed Switch

Magnetic proximity sensor used to detect door status, triggering alerts if the unit remains unsealed.
